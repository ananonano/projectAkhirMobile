## Booking Code Format

### Format Specification
- **Prefix**: `BKG` (Booking)
- **ID**: 5-digit zero-padded number
- **Example**: `BKG00001`, `BKG00042`, `BKG12345`

### Generation Logic
```dart
final bookingCode = widget.bookingId != null 
    ? 'BKG${widget.bookingId.toString().padLeft(5, '0')}' 
    : 'BKG00000';
```

### ID Range
- Minimum: `BKG00001` (first booking)
- Maximum: `BKG99999` (99,999 bookings)
- After 99,999: `BKG100000` (6 digits, still works)


### Database Table
```sql
CREATE TABLE bookings (
  id INTEGER PRIMARY KEY AUTOINCREMENT,  -- ← Source of booking ID
  user_id INTEGER,
  lapangan_id INTEGER,
  nama_lapangan TEXT,
  tanggal TEXT,
  jam TEXT,
  total_harga TEXT,
  status TEXT DEFAULT "completed",
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
```
